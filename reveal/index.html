1 . As usual, there are a variety of editions of this Windows OS. How many different editions of Windows 7 are there?

    1. Answer - Six

    Names -
  1. Windows 7 Ultimate
  2. Windows 7 Starter
  3. Windows 7 Professional
  4. Windows 7 Enterprise
  5. Windows 7 Home Premium
  6. Windows 7 Home Basic

2. Windows 7 has a variety of accessories included on the Start menu. However, most are also included with the previous Windows Vista. Which of these accessories is new to Windows 7?

The Snipping Tool, Sticky Notes , Paint , Run

2. Answer - Sticky Notes

Sticky Notes is one of the new Windows 7 accessories. With it you can stick notes to the desktop, like virtual sticky notes.
3. The Windows XP Mode ensures greater backward compatibility with previous software. However, it will only run on a few editions of Windows 7. Which Windows 7 editions include Windows XP Mode?
3. Answer -
Windows 7 Ultimate,Windows 7 Professional & Windows 7 Enterprise
4. The Windows 7 Media Center brings together a variety of Windows 7 multimedia tools. However, one edition of Windows 7 does not include the Media Center. Which edition is that?

4. Answer- Windows 7 Starter

Windows 7 Starter is the most basic edition of Windows 7, and so does not include the Windows 7 Media Center. Although it still includes most of the bundled applications, and so is more suitable for netbooks.

5. Windows 7 includes a variety of bundled software applications. Which one of the following software applications DOES NOT come included with Windows 7?

Internet Explorer 9 , Media Player 12 Windows Live Photo Gallery , Paint

5. Answer - Internet Explorer 9
6. Windows 7 system requirements include a minimum allocation of RAM. So, what is the minimum RAM required for Windows 7?
6. Answer- 1 GB

7. Windows 7 also includes a number of system tools. Which of the following IS NOT a system tool included with Windows 7?

MSCONFIG , CCleaner , Task Scheduler , System Restore

7. Answer- Ccleaner

CCleaner is not a system tool included with Windows 7. However, it is freeware software which is worth adding.

8. Windows 7 has plenty of shortcut keys. Which of these is the shortcut key for opening Jump Lists on the taskbar?

Ctrl + Shift + N , Alt + Win + Number (1-9)

8. Answer- Alt + Win+ Number(1-9)

Alt + Win + number (1-9) is the shortcut key that will open taskbar Jump Lists. Press a suitable number from 1 - 9 depending on the position of the application icon. So, the furthest left will be one and the furthest to the right will be a highest number, depending on how many application shortcuts are on the taskbar.

9. DirectX is the set of Microsoft extensions that maximize graphic hardware. There are various versions of DirectX, and which one of these versions was introduced with Windows 7?

DirectX 10 , DirectX 12, DirectX 11 , DirectX 11.1

9. Answer- DirectX 11

DirectX 11 is the version for Windows 7. The next version will be DirectX 11.1 for Windows 8.
10. Windows 7 has a show desktop option that minimizes all open windows. Where is this option included?

10. Answer- Taskbar

The show desktop option is included on the far right of the taskbar. When selected all the open windows will be minimized. Although when you move the cursor away from show windows they will be maximized again.

11. Windows 7 has a number of exciting themes to select from. Which of these IS NOT a Windows 7 theme?

United Kingdom , Sport , Landscape

11.Answer- Sport
12. Windows 7 Aero has been enhanced with what?

12.Answer - Aero Shake

Windows Aero Shake is something that was not included with Windows Vista Aero. As such, shake the window with the mouse and all the other open windows will be minimized to the taskbar.
13. Lots of software is compatible with Windows 7.

However, some software is not. Which of these software packages WILL NOT run on Windows 7?

iWorks , OpenOffice , Safari , iTunes

13. Answer- iWorks

iWorks is Apple's office suite that is exclusively for the Apple Mac OS. Other Apple software such as Safari and iTunes can be installed on Windows 7 as well as the Mac OS. OpenOffice is a mutliplatform office suite for Windows 7 and other platforms.
14. Which Windows 7 does not include Media Player 12?

14. Answer- Windows 7 N

Windows 7 N is a sub-edition of Windows 7 that does not include Media Player 12 or other associated software such as the Media Center. However, aside from this it has everything else you would expect to find with Windows 7.

15. Which of the following will you not find included with Windows 7 Starter?

Gadgets , Windows Fax And Scan , Windows Flip 3-D

15. Answer- Windows Flip 3-D

Much of Windows 7 Aero is not included with Windows 7 Starter. As such, Windows Flip 3-D is something that is not included with Windows 7 along with Aero Glass, Aero Peek and Aero Shake.
